# Monika Cloud TOS


为保证服务可用性，我们调整了Monika Cloud的使用条例。

1. Monika Cloud（简称莫盘）是一个为DDLC内容作者提供的存储平台。
2. 你不应该在莫盘储存个人信息，因为有被泄露的风险。
3. 莫盘不是公共网盘，任何数据都可能失效而不能保存。
4. 不应上传含违反《中华人民共和国著作权法》的内容。
5. 莫盘初始用户空间将被调整为5GB。
6. 请勿违反所在地法律。
7. 如您不慎违反以上条例或者不承认以上条例，我们有权暂停您的账户的使用。
